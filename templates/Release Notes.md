SLE Manager Release Notes 
Version: Raven 2025 27
Release Date: 12.17.2025

**New Features:**
- Additional audio languages are now published to Freewheel to enable Spanish language ads targeting
- SLES now can have multiple sports events created for them to support Winter Olympics broadcasts that contain multiple separate sporting events
- Added DolbyVision transcoding profiles
- SLEM UI web app framework has been upgraded, providing many visual and user experience enhancements, improving readability, consistency, and performance.
- SLEM supports publishing prefetch schedules for large scale DAI
  
**Improvements:**

- Support for automated creation of Winter Olympics SLE via SWIM
- Updated playback URL shims to identify cluster
- SLEM UI displays video quality for each encoder on the event details page
- SLEM UI has a dark mode! Click your username in the top right and click the moon/sun icon to change. 

**Bug Fixes:**

- Fixed an edge case issue with publishing expired replays to Merlin
